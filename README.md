# 🛍️ E-Commerce Recommendation System

A modular, production-ready **Content-Based Recommendation System** built using:

- TF-IDF Vectorization
- Cosine Similarity
- Scikit-Learn
- Modular ML Pipeline Structure

This project follows industry-standard ML project architecture with proper:
- Data ingestion
- Data preprocessing
- Feature engineering
- Model training
- Logging
- Custom exception handling

---

## 📌 Problem Statement

To build a content-based recommendation system that suggests similar products based on product metadata such as:

- Category
- Sub-category
- Brand
- Description

The system generates recommendations using text similarity techniques.
